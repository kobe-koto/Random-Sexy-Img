...和API!

我使用 Cloudflare Workers 提供了一個隨機色圖的 API, 下面是個傳回數值的定義.

一共有以下幾個傳回值.

返回數值名描述示例
code請求狀態碼200
msg請求信息OK
type請求的數據庫fur或gay
count指示數據庫收入了多少張圖片1000
name圖像檔案名IMG_0176-1.JPG
url指向圖像的連結https://drive-koto.vercel.app/api/raw/?path=/Image/GetColorImg/fur/IMG_0176-1.JPG
urlPreview指向圖像的預覽用鏈接https://drive.koto.cc/Main/Image/GetColorImg/fur/IMG_0176-1.JPG?preview

返回的數據為JSON格式, 示例如下

{"code":"200","msg":"OK","type":"fur","count":"1078","name":"IMG_0176-1.JPG","url":"https://drive-koto.vercel.app/api/raw/?path=/Image/GetColorImg/fur/IMG_0176-1.JPG","urlPreview":"https://drive.koto.cc/Main/Image/GetColorImg/fur/IMG_0176-1.JPG?preview"}

要調用此 API - fur Edition, 發送 Get 請求到 rsi.koto.cc/fur?type=json .

要調用此 API - gay Edition, 發送 Get 請求到 rsi.koto.cc/fur?type=json .

另注: API允許跨域請求(Access-Control-Allow-Origin: *), 禁止緩存(Cache-Control: no-store).

在這裏面的FurUpload或者GayUpload上傳您的色圖!
